NFL FootBall Teams

Below you will find some stats based on two NFL Teams. I made both the teams scores per quarter and total for both teams each quarter. Then the final Scores at the end. This project was rather difficult to get started as I had to figure out how to use tables but once I did it was significantly easier to finish the rest of this.

HTML is a language I could see myself using on a daily basis and I think I made the right career decision.

Towards the end of the project I thoroughly enjoyed using HTML. Thanks for checking out my first HTML Project!
